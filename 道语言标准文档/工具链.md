# 工具链

## 一、编译器

### 1. 编译命令
```bash
# 基本编译
道 编译 源码.dao

# 指定输出
道 编译 -出 程序 源码.dao

# 交叉编译
道 编译 -系统 linux -构架 amd64 源码.dao
```

### 2. 编译选项
```bash
# 优化级别
道 编译 -优 2 源码.dao

# 调试信息
道 编译 -调 源码.dao

# 静态链接
道 编译 -静态 源码.dao
```

## 五、文档工具

### 1. 文档生成
```bash
# 生成文档
道 档 包名

# 启动文档服务
道 档 -服务 -端口 6060
```

### 2. 文档注释
```go
// 包 计算 提供基本的数学计算功能
包 计算

// 加法 计算两数之和
//
// 入：
//   - 甲：第一个数
//   - 乙：第二个数
// 还：
//   两数之和
术 加法(甲 整, 乙 整) 整 {
    还 甲 + 乙
}
```
