## 快速开始

### 安装
```bash
go install github.com/dao-lang/dao@latest
```

### 你好，世界！
```go
包 主

引 (
    "标准/示"
)

术 主() {
    示("你好，世界！")
}
```

### 运行
```bash
dao run 你好.dao
```
